# Критические исправления v1.0.1

## Исправленные ошибки

### 1. AttributeError с InlineKeyboardButton

**Проблема:** `AttributeError: 'CallbackQuery' object has no attribute 'types'`

**Решение:** Исправлены все случаи неправильного использования `callback.types.InlineKeyboardButton` в файлах:
- `admin/handlers.py` - 5 случаев
- `bot/additional_handlers.py` - 4 случая

Теперь используется правильный импорт: `from aiogram.types import InlineKeyboardButton`

### 2. Отсутствующий обработчик настроек

**Проблема:** Кнопка "Настройки" не работала

**Решение:** Добавлен обработчик `show_settings()` для callback_data "settings" в `bot/additional_handlers.py`

### 3. Истечение кода подтверждения Telethon

**Проблема:** `The confirmation code has expired (caused by SignInRequest)`

**Решение:**
- Добавлено предупреждение о том, что код приходит в Telegram (не по SMS)
- Улучшена валидация кода (только 5 цифр)
- Добавлена обработка ошибки истечения с понятным сообщением
- Добавлены try-except блоки для всех операций с Telethon

### 4. Отсутствующий обработчик информации о режимах

**Проблема:** Кнопка "О режимах" не работала

**Решение:** Добавлен обработчик `show_mode_info()` для callback_data "mode_info"

## Инструкции по развертыванию

### Быстрая установка

1. Распакуйте архив:
```bash
unzip merai_monitoring_fixed.zip
cd merai_monitoring
```

2. Установите зависимости:
```bash
pip install -r requirements.txt --break-system-packages
```

3. Настройте конфигурацию в `config.py`:
```python
BOT_TOKEN = "ваш_токен"
TG_API_ID = ваш_api_id
TG_API_HASH = "ваш_api_hash"
ADMIN_ID = ваш_telegram_id
```

4. Запустите бота:
```bash
python3 main.py
```

### Решение проблемы с кодом Telethon

**Важно:** Код подтверждения приходит в сообщении Telegram, а не по SMS!

Если вы видите ошибку "The confirmation code has expired":

1. Начните процесс заново через меню Dual Mode
2. После ввода номера телефона НЕ ЖДИТЕ SMS
3. Проверьте сообщения от Telegram в приложении
4. Введите код в течение 5 минут
5. Формат кода: 5 цифр (например, 12345)

### Тестирование функциональности

После запуска протестируйте:

1. **Основное меню:**
   - Отправьте `/start` боту
   - Проверьте что все кнопки работают

2. **Настройки:**
   - Нажмите "Настройки"
   - Убедитесь что открывается меню

3. **Dual Mode:**
   - Попробуйте переключить режим
   - Проверьте информацию о режимах

4. **Админ панель (только для администратора):**
   - Откройте админ панель
   - Проверьте список пользователей
   - Попробуйте просмотреть детали пользователя

5. **Подписки:**
   - Откройте меню подписок
   - Проверьте отображение тарифов
   - Попробуйте активировать пробный период

## Известные особенности

### Подключенные боты

Функциональность подключения ботов работает следующим образом:
- Пользователь отправляет токен своего бота
- Бот верифицирует токен через Telegram API
- Пользователь получает +7 дней подписки
- Токен сохраняется в БД

**Примечание:** Подключенные боты не запускаются автоматически. Эта функциональность будет добавлена в следующей версии.

### Business API ограничения

Business API не может:
- Сохранять сообщения с таймером самоуничтожения
- Сохранять сообщения с лимитом на 1 просмотр
- Работать с секретными чатами
- Отслеживать удаление всего чата

Для обхода этих ограничений используйте Userbot или Dual Mode.

## Техническая поддержка

При возникновении проблем:

1. Проверьте логи в `logs/merai.log`

3. Проверьте версию Python (требуется 3.11+)
4. Убедитесь что все зависимости установлены
