# 🚀 Быстрый старт MerAI & Monitoring

Данное руководство предоставляет пошаговые инструкции для быстрого запуска системы мониторинга.

## Предварительные требования

Перед началом установки убедитесь, что у вас установлен Python версии 3.11 или выше. Проверить версию Python можно командой `python3 --version`. Также необходим токен Telegram бота, который можно получить у BotFather через команду /newbot.

Для работы Userbot режима потребуются Telegram API credentials. Получить их можно на сайте my.telegram.org в разделе API development tools. После регистрации приложения вы получите API ID и API Hash.

## Установка за 5 минут

Начните с клонирования или загрузки проекта в выбранную директорию. Перейдите в директорию проекта через командную строку.

Установите все зависимости командой:
```bash
pip install -r requirements.txt --break-system-packages
```

Параметр break-system-packages необходим для установки в системное окружение Python. При работе в виртуальном окружении этот параметр можно опустить.

## Настройка конфигурации

Откройте файл config.py в текстовом редакторе. Найдите раздел ОСНОВНАЯ КОНФИГУРАЦИЯ в начале файла.

Замените значение BOT_TOKEN на ваш токен, полученный от BotFather. Убедитесь, что токен указан в кавычках.

Для настройки Userbot режима укажите TG_API_ID и TG_API_HASH, полученные на my.telegram.org. API ID должен быть числом без кавычек, а API Hash строкой в кавычках.

Установите ваш Telegram User ID в переменную ADMIN_ID. Узнать свой ID можно через бота @userinfobot.

Альтернативным способом конфигурации является использование файла .env. Скопируйте файл .env.example в .env командой `cp .env.example .env` и отредактируйте его значения. Система автоматически загрузит переменные из .env при запуске.

## Первый запуск

Запустите систему командой:
```bash
python3 main.py
```

При успешном запуске вы увидите логотип MerAI & Monitoring и информацию о подключении бота. Система автоматически создаст базу данных в директории database/merai.db и все необходимые таблицы.

Проверьте работу бота отправив ему команду /start в Telegram. Вы должны получить приветственное сообщение с основным меню.

## Подключение Business API

Для использования Business API режима требуется активная подписка Telegram Premium на вашем аккаунте. Откройте настройки Telegram и перейдите в раздел Business Tools или Бизнес-инструменты.

Добавьте вашего бота в список бизнес-инструментов через поиск по username. После подключения бот автоматически начнет мониторинг сообщений в ваших бизнес-чатах. Вы получите трехдневный пробный период для тестирования всех функций.

## Настройка Userbot режима

Для активации Userbot режима откройте меню бота и выберите раздел Dual Mode. Нажмите на кнопку Userbot для переключения в этот режим.

Система запросит номер вашего телефона в международном формате, например +79991234567. После отправки номера вы получите код подтверждения в сообщении Telegram от Telegram самого. Важно отметить, что код приходит именно в Telegram, а не через SMS на телефон.

Введите полученный код в формате 12345 без пробелов и дефисов. После успешной авторизации Userbot начнет мониторинг всех ваших чатов автоматически.

## Проверка работы

Протестируйте систему отправив несколько сообщений в тестовый чат, затем удалите одно из них. Бот должен прислать уведомление об удаленном сообщении с его содержимым.

Отредактируйте существующее сообщение в чате. Система зафиксирует изменение и покажет оригинальный и новый текст в формате «Было/Стало».

Проверьте детектор контактов отправив сообщение с телефоном, email или ссылкой. Система автоматически извлечет и сохранит обнаруженные контакты.

## Управление подписками

После окончания пробного периода выберите подходящий тариф в меню Подписки. Оплата осуществляется через Telegram Stars с автоматической конвертацией в рублевый эквивалент.

Базовый тариф предоставляет мониторинг до 1000 сообщений на 30 дней за 50 Stars. Премиум тариф обеспечивает безлимитный мониторинг и все дополнительные функции за 150 Stars. Lifetime подписка предлагает пожизненный доступ ко всем функциям за 500 Stars.

Получите бонус семь дополнительных дней подписки создав нового бота через BotFather и подключив его токен через соответствующую кнопку в меню.

## Развертывание на BotHost

Для развертывания на платформе BotHost создайте новый проект в панели управления. Загрузите все файлы проекта через git репозиторий или файловый менеджер.

В настройках проекта выберите Python 3.11 или выше как runtime. Установите зависимости через requirements.txt в разделе управления пакетами.

Настройте переменные окружения в соответствующем разделе панели управления. Укажите BOT_TOKEN, TG_API_ID, TG_API_HASH и ADMIN_ID.

Установите точку входа как main.py в настройках запуска проекта. Запустите проект через кнопку Start или Deploy.

## Решение типичных проблем

При ошибке подключения к Telegram API проверьте корректность токена бота и доступность интернет-соединения. Убедитесь, что бот не заблокирован в Telegram.

Если Userbot не авторизуется убедитесь, что API ID и API Hash указаны корректно. Проверьте, что код подтверждения вводится без пробелов и дополнительных символов. Попробуйте запросить новый код, если предыдущий истек.

При проблемах с сохранением сообщений проверьте наличие активной подписки и её срок действия. Убедитесь, что не достигнут лимит сообщений для текущего тарифа. Проверьте логи на наличие ошибок базы данных.

Если база данных не создается автоматически проверьте права доступа к директории database. Убедитесь, что SQLite3 установлен в системе. Попробуйте создать директорию database вручную.
